JAVASCRIPT이란 무엇인가?

Javascript is the programming language of HTML and the Web.

자바스크립트가 html을 제어하는 언어이다.
script 태그 안쪽의 코드를 javascript로 인식한다.
onclick 속성
onclick이라는 속성의 값은 반드시 javascript가 와야한다.
onclick 속성의 속성값은 웹브라우저가 기억하고 있다가 그 속성이 위치한 곳을
사용자가 클릭했을때 js의 문법에 따라 웹브라우저가 동작
웹브라우저에서 일어나는 일들을 사건, 영어로는 event라고 한다.
on으로 시작하는 속성들
onclick, onchange(값이 바뀐 경우),
onkeydown(키를 누른 경우- javascript onkeydown event attribute검색)
