Javascript is the programming language of HTML and the Web.

자바스크립트는 html을 제어하는 언어이다. html은 한번 선택하면 바뀌지 않는 정적인 언어.

자바스크립트의 6가지 데이터 타입
1. Boolean
2. Null
3. Undefined
4. Number
산술 연산자 : + - * /
3+3, 3-2, 2*2, 6/3
5. String
"hello world", "hello world".length,
문자 사용할 때 쓰이는 명령들(javascript string 구글링) [사이트]
1+1 => 2, "1"+"1" => "11"
6. Symbol

변수(variable) vs 상수(constant)
var a = 1 -> 이렇게 var을 써준다.
1 = 2 (error!)
변수를 사용하는 이유: 바꾸어야하는 부분을 하나로 통일하여 한 번에 바꿀 수 있다.

document.querySelector('body')//tag
document.querySelector('.myclass')//class
document.querySelector('#idname')//id

querySelector는 해당 코드가 처음 등장하는 부분만 가져온다.
document.querySelector('a') 는 a태그가 가장 먼저 등장하는 부분 한 개만 가져온다.
