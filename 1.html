HTML이란 무엇인가?

Hypertext Markup Language(HTML) is the standard markup language for creating web pages and web applications.

1. tag : like strong, /strong covered in brackets
strong tag : to mark as bold
underline tag : to make words underlined
header tag : h1 - h6 gets smaller word size
new line tag : br tag
paragraph tag : p tag
image tag : img
list tag : li tag -> abrev of list
ul tag : parent tag of li that we must include to use li tag. = unordered list
ol tag : numbering each list. It is the parent tag of li. = ordered list
a tag : 가장 대표적인 태그. 이 태그 덕분에 웹페이지를 발견하고 최고의 검색 결과를 만들 수 있다. link(hypertext)를 위한 태그 = anchor

2. attribute : like src, width to give more information in tag
