CSS란 무엇인가?

Cascading Style Sheet(CSS) is a language that describes the style of an HTML document. CSS describes how HTML elements should be displayed.

css : html과 분리된, 디자인에 관여하는 것.

1. style tag 안에 css 문법을 넣어준다.

-> font로 각 글자의 색깔을 표현하는 대신, 헤드의 style 태그 안에 해당 태그 부분의 스타일을 지정하여 표현한다.
head에 안에 어떤 부분에 css를 적용할지 명시하는 a{}와 같은 선택자 selector를 적어준다.
그리고 그 안에는 color:red와 같이 한국어로는 '효과', 영어로는 'declaration'을 의미하는 값을 넣어준다.
color는 property, red는 property value라고 한다.
여러 효과를 넣어줄 수 있으며 이때에는 세미콜론(;)을 이용하여 구분해준다.
효과 : 중복 제거, html을 이용하여 디자인하는 것보다 훨씬 효과적, 디자인과 관련된 태그를 따로 분리함으로써 정보만을 가지고 있는 코드를 분석하기가 용이하다.

2. style 속성을 이용한다.

-> 디자인을 지정하려는 태그 부분에 속성 style을 넣어 값을 지정한다. ex. style="color:red"
style 부분은 html의 속성이며 그 값으로 css의 효과가 들어온다.
웹브라우저는 css의 문법에 따라 해석해서 그 결과를 style 속성이 위치하고 있는 태그에 적용할 것이다.
만약 head 안의 style에 해당 태그에 모두 적용한 후 그 중 하나의 코드에만 다른 css 효과를 적용한다면
style 속성을 이용한다. ex. text-decoration:underline
여러 개 적용할 때에는 style="color:red;text-decoration:underline"

3. 클래스를 지정한다.

-> 원하는 태그에 class 속성을 추가하여 class 이름을 지정해준다.
그리고 head 내부에 .(class 이름){}를 적어서 효과를 지정한다.
이때 .(class 이름)은 해당 class 이름을 가진 태그들에게 {}안의 효과를 적용하겠다는 의미이다.
만약 해당 클래스 내에서도 특정 태그는 다른 효과를 적용하고 싶다면 class="class1 class2"로 지정하여
head에 다른 효과를 지정해서 명시해주면 된다.
이때 head에서 상하 위치가 중요한데 아래에 있을 수록(==보다 가까이 있는 것이) 우선 순위가 높다.

4. id 선택자를 사용한다.

** 우선순위 : 태그 선택자 < 클래스 선택자 < 아이디 선택자 -> 같은 선택자인 경우 아래에 있을수록, 더 가까울수록 우선순위가 높다.
코드를 작성할 때 클래스는 포괄적인것, 아이디는 구체적인 예외처리를 할때 사용하는 경우가 많다.
id 선택자는 head 안에 #(id이름){}으로 효과를 지정해주며 위치에 관계없이
가장 높은 우선순위를 갖습니다.
id 선택자 이름은 한 번이라도 사용된 경우, 중복되어 더 이상 등장할 수 없다.
-> 이 문서에서는 유일무이한 값. 따라서 a 태그와 같이 상대적으로 포괄적인 값에 비해 훨씬 구체적이고 범위가 좁다.
범위가 좁을 수록 더 우선순위가 높다고 기억하기**

< box model >

참고 - css box model

1. block level element(==tag)
- h1 tag, div tag, ol tag의 경우, 한 줄을 차지한다.

2. inline level element
- a tag, span tag의 경우, 글자 만큼의 자리를 차지한다.

h1과 a 태그의 효과가 같은 경우, h1,a{}로 묶어서 나타낸다.
border-width, border-color와 같이 border가 반복되는 경우, border:5px solid red
로 순서에 상관없이 나열하면 된다.

box와 컨텐츠의 여백을 주기 위해서는 padding을 사용한다.
테두리와 테두리 사이 간격 지정할 때는 margin을 사용한다.
너비는 width, 높이는 height로 지정한다. border-right, border-bottom과 같은 효과를 적용하고 마우스 오른쪽-> 검사 에서
조정이 필요한 값 margin, padding 등을 조절한다.
display 속성은 none, block, inline 등의 값을 넣을 수 있다.
기본 태그로 div(block level), span(inline level) 태그 사용 가능.

< 코드의 재사용성 >

각 html 파일마다 css 코드를 넣으면 웹페이지를 하나 추가하거나, css 코드를 수정할 때,
많은 시간이 걸리고 비효율적인 작업이 반복된다.
따라서 style.css라는 파일을 따로 생성한 후, 각 html 파일에
link rel="stylesheet" href="style.css"을 < >안에 적어서 head안에 넣으면 동일하게 작동된다.
-> 중복을 제거해서 재사용성을 높이고 가독성을 높이고 유지보수를 편하게 한다.
-> 이렇게 되면, 각 웹페이지를 로드할 때마다 style.css 파일을 다운로드하게 되므로 트래픽이 많이 사용되지 않나?
no. 캐시를 통해 한 번 다운로드 받은 파일은 컴퓨터에 저장해두었다가 필요할 때 다시 가져온다.
저장된 결과를 가져오기 때문에 속도도 빠르고, 트래픽 사용료도 훨씬 적게 낼 수 있다.
